# 🇬🇧 English Practice Portal - AI Tutor

An interactive web application designed for English language students (Levels A1-A2). This portal transforms static educational content into a dynamic learning experience, allowing students to practice grammar, speaking, and writing in a single unified environment.

### ✨ Key Features
- **Dynamic Theory & Practice:** Automatically parses lessons from text files, organizing them into interactive modules with grammar structures and "Karen's Tips".
- **AI Speaking Center:** Integrated Text-to-Speech (TTS) for pronunciation modeling and a voice recorder for student practice.
- **Smart Exercise Engine:** Generates "Fill in the blanks" and translation drills with real-time feedback based on lesson materials.
- **Writing Module:** A dedicated space for students to submit and save writing assignments directly to the server.

### 🛠️ Tech Stack
- **Python:** Core logic and data processing.
- **Streamlit:** Web framework for the interactive UI.
- **gTTS (Google Text-to-Speech):** High-quality voice synthesis.
- **Streamlit Mic Recorder:** Audio capture for speaking assessments.

### 🚀 How to Use
1. Add your lesson materials in the `Clases/` directory following the standard format.
2. Run the application using `streamlit run app.py`.
3. Select a lesson from the sidebar and start practicing!
